## 🎨 设计理念

### 核心原则
- **视觉层次**：清晰的视觉层次结构，引导用户注意力
- **微交互**：精致的悬停效果和过渡动画，提升交互体验
- **现代感**：玻璃态、渐变、模糊效果等现代设计元素
- **响应式**：完美适配各种屏幕尺寸
- **性能优化**：流畅的动画和过渡效果

### 设计语言
- **Glassmorphism（玻璃态）**：半透明背景 + 背景模糊
- **Neumorphism（新拟态）**：柔和的阴影和光泽效果
- **Gradient（渐变）**：优雅的多彩渐变装饰
- **Motion Design（动效设计）**：有意义的动画反馈

#### 2. **文字渐变**
```css
bg-gradient-to-r from-foreground to-foreground/70
bg-clip-text text-transparent
```

#### 3. **边框渐变**
```css
border-primary/20  /* 20% 透明度 */
border-border/50   /* 50% 透明度 */
```

## 📐 布局系统

### 响应式断点
```
- sm: 640px   (小屏幕)
- md: 768px   (中屏幕)
- lg: 1024px  (大屏幕)
- xl: 1280px  (超大屏幕)
```

### 容器宽度
```tsx
container mx-auto px-4 lg:px-8  /* 响应式内边距 */
max-w-7xl                       /* 最大宽度限制 */
```

### 网格布局
```tsx
grid gap-4 sm:grid-cols-2 lg:grid-cols-3  /* 响应式列数 */
```

### 1. **动画性能**
- 使用 `transform` 和 `opacity` 而非 `width/height`
