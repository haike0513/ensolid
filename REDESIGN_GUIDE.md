### 色彩方案
```
主色系:
- Primary → Blue-600
- Primary → Cyan-500
- Foreground → Primary → Foreground

技术栈:
- Blue-600 → Cyan-600 (SolidJS)
- Blue-700 → Blue-500 (TypeScript)
- Purple-600 → Yellow-500 (Vite)
- Orange-600 → Amber-500 (pnpm)

特性卡片:
- 6种不同的渐变主题
```

### 动画系统
```
入场动画:
- fade-in-up: 向上淡入
- fade-in-down: 向下淡入
- scale-in: 缩放入场
- animate-pulse: 脉冲动画

悬停动画:
- translate: 位移
- scale: 缩放
- rotate: 旋转
- opacity: 透明度
- shadow: 阴影增强
```

## 📝 开发笔记

### 关键设计决策

**1. 为什么使用玻璃态？**
- 现代感强
- 层次分明
- 视觉通透

**2. 为什么使用渐变？**
- 视觉丰富
- 品牌识别
- 引导视线

**3. 为什么添加光晕？**
- 科技感
- 深度感
- 吸引注意

**4. 为什么序列动画？**
- 节奏感
- 层次感
- 不突兀

### 性能考虑

```
✅ 使用transform而非position
✅ 使用opacity而非visibility
✅ GPU加速动画
✅ 合理的动画时长
✅ 避免频繁重排重绘
```

### 可访问性

```
✅ 语义化HTML
✅ ARIA标签
✅ 键盘导航
✅ 对比度优化
✅ 响应式设计
```

## 🔧 自定义建议

### 颜色定制
修改 `src/index.css` 中的CSS变量：
```css
:root {
  --primary: oklch(0.205 0 0);
  --primary-foreground: oklch(0.985 0 0);
  /* 更多变量... */
}
```

### 动画调整
修改组件中的动画延迟和持续时间：
```tsx
style={{ "animation-delay": "0.1s" }}
class="transition-all duration-500"
```

### 间距调整
修改Tailwind类名：
```tsx
class="py-20 sm:py-32"  // 调整垂直间距
class="gap-6"           // 调整元素间距
```
